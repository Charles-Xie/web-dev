AJAX

AJAX

AJAX == HTTP Request。

AJAX (Asynchronous JavaScript and XML) 就是 HTTP 网络请求。早年 Web 没有 AJAX 时, 通常是整体刷新页面的, 局部刷新需要通过 iframe 等 tricky 方式实现。AJAX 出现后以 Gmail 为代表的 Web App 触发了大规模的页面体验改造(史称"Web 2.0")。
